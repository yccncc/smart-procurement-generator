# AI政府採購｜採購智作

以結構化需求訪談、官方範本管理、AI 協作起草及跨文件一致性檢核，協助政府機關承辦人製作採購文件的展示原型。

## 線上展示

GitHub Pages 啟用後，網站預計位於：

https://yccncc.github.io/ai-government-procurement/

## 原型功能

- 引導式採購需求輸入
- 參考範本及歷史案件選擇
- 採購規劃摘要及待確認事項
- 多份採購文件草稿生成
- 跨文件一致性與風險檢核
- 列印及 HTML 文件包下載

## 使用限制

本網站為產品流程展示原型，尚未連接官方範本版本庫、法規規則庫及正式 AI 服務。產出內容須由權責人員審核，不得直接作為正式招標文件。
